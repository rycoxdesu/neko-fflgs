# Neko's FastFlags Saver

Aplikasi ini memungkinkan Anda untuk menyimpan dan mengelola FastFlags Roblox dengan mudah.

## Fitur
- Menyimpan konfigurasi FastFlags/JSON ke file IxpSettings.json
- Preset untuk Lag-Ball dan SmoothBall
- Antarmuka yang modern dan mudah digunakan
- Tampilan side-by-side untuk input dan tombol aksi

## Persyaratan Sistem
- Windows 7 atau lebih baru
- Roblox harus ditutup sebelum menggunakan aplikasi

## Cara Menggunakan
1. Jalankan aplikasi
2. Paste FastFlags/JSON Anda ke dalam kotak teks
3. Klik "Apply" untuk menyimpan pengaturan
4. Buka Roblox untuk menerapkan perubahan

## Build Aplikasi
Lihat file BUILDING.md untuk instruksi lengkap tentang cara membuat build aplikasi.
